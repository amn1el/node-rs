# node-rs
A collection of utility packages written in Rust using [napi-rs](https://napi.rs/), designed for private use in my [Node.js](https://nodejs.org) projects.

## Packages
- **[humanize](./packages/humanize)**: A Rust package leveraging napi-rs to convert durations (in milliseconds) and byte sizes into human-readable formats.
- **[i18n](./packages/i18n)**: Utilities for internationalization, including message translation and formatting.
- **[security](./packages/security)**: Provides AES-256 encryption and decryption, along with a utility for cyclic number reversal.
